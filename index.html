Dynamic Full-Stack Website

This is a dynamic website built with inline HTML, CSS, and JavaScript that mimics a GitHub repository interface.

Features:

  • GitHub-like UI with dark theme
  • Interactive file management
  • Watch/Star/Fork functionality
  • Search through files
  • Add/edit/delete files dynamically
  • Modal dialogs for actions
  • Toast notifications

How to Use:

  1. Click "Add file" to create new files
  2. Use the search box to filter files
  3. Click on file names to view/edit them
  4. Use the Watch/Star/Fork buttons
  5. Click "Code" button to clone the repo

Technologies Used:

  • HTML5
  • CSS3 (Inline)
  • JavaScript (Inline)
  • Font Awesome Icons
